Why Choose
in Your Classroom?

EduHeLx facilitates instruction and learning in a cloud-native, highly scalable, virtual classroom environment by providing customizable computing environments for data science courses.

Customized for your needs.

Students learn to use data science tools popular in industry and academic research including Jupyter notebooks and RStudio. Instructors are able to concentrate on pedagogical material without worrying about the computing infrastructure.

Fantastic Features

EduHeLx is customized for the unique needs of your classroom, your materials, and your students.

Customized Scaling

Each EduHeLx class is deployed using Docker, Kubernetes, and Helm, meaning ease of access to course materials regardless of class size

Flexible Data Access

Class data can be shared internally as well as externally. EduHeLx also allows instructors to store files in a shared class folder for access by all students

Launch Only What You Need

EduHeLx tools are containerized applications. Our standard EduHeLx suite provides access to Jupyter notebooks, RStudio, a virtual desktop, and file browser - but custom apps can be added to your instance to support your unique classroom needs

Cloud Native, Lightweight Orchestration

As a low-code, dynamically orchestrated, metadata driven framework, EduHeLx provides instructors and students a uniquely no-download, low-barrier entry to their classroom environment
